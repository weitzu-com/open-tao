# Contributing

Thanks for helping with Open Tao.

## Contribution rules
- Use public-domain Chinese text.
- English translations must be *your original work* (or AI-assisted and edited by you), not copied from existing copyrighted translations.
- By contributing, you agree to release your contributions under CC0 1.0 Universal (public domain).

## Recommended structure
- `english/tao_te_ching_en.md`: full English translation.
- `chapters/NN.md`: bilingual chapter files (Chinese + English).
- `translations/<language>/`: additional languages (optional).

## How to contribute
1. Fork the repo and create a branch.
2. Add or update the relevant files.
3. Submit a Pull Request with what you changed and why.
