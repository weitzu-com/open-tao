# Translations

This directory hosts *non-English* translations for the Open Tao project.

## Rules
- Only contribute translations you authored yourself.
- Do **not** copy from copyrighted translations.
- Contributions are dedicated to the public domain under **CC0 1.0**.

## Structure
- Create a subfolder per language, using an ISO 639-1 code when possible.

Examples:
- `translations/es/tao_te_ching_es.md`
- `translations/fr/tao_te_ching_fr.md`
- `translations/ja/tao_te_ching_ja.md`

You may also add chapter files if you prefer, but please keep naming consistent.
